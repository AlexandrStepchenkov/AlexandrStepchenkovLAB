Программа: Каршеринг

Основные сущности:
- User (userId, name, email, phone)
- Car (carId, model, licensePlate, pricePerKm, available)
- Ride (rideId, userId, carId, status, distanceKm, durationHours)
- Payment (paymentId, rideId, amount)

## Программа умеет создавать, просматривать, изменять и удалять каждую из сущностей.

Также реализована бизнес-логика:
- начало поездки (машина становится недоступной),
- завершение поездки (машина возвращается, рассчитывается стоимость, создаётся платёж),
- список свободных машин,
- история поездок пользователя,
- общая выручка сервиса,
- оплата по конкретной поездке,
- список всех активных и завершённых поездок.
